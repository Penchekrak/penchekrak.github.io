Conclusion

  • We have shown that it is natural and beneficial to apply topological regularizations in some domains, if one has meta information or wants to impose inductive bias on structures that occur.
  • Author believes that proposed regularization for point cloud regression/optimization may be quite useful if equipped within correct setup and it’s best performance regimes are yet to be found/tuned.
  • Employment of higher dimensions of homologies still remains somewhat a challenge, since one needs a parallel algorithm with feasible computation time and memory requirements.
